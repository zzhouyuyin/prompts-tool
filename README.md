# prompts-tool
A collection of high-quality ChatGPT prompts for daily use
# ChatGPT 高质量提示词工具
一个整理了日常工作、学习、编程场景的 ChatGPT 提示词合集，帮助提升使用效率。
## 场景分类
### 1. 编程辅助
请作为资深 Python 工程师，帮我检查以下代码的语法错误和逻辑漏洞，并给出优化建议：
[在这里粘贴你的代码]
### 2. 文案写作
请以「AI 工具使用技巧」为主题，写一篇 500 字左右的科普短文，语言通俗易懂，适合新手阅读。
### 3. 学习总结
请用思维导图的形式，总结「Python 基础语法」的核心知识点，分点清晰，突出重点。
## 使用说明
1. 复制对应场景的提示词
2. 粘贴到 ChatGPT 对话框中
3. 根据自己的需求修改括号内的内容
